A **private, interactive web app** built with Streamlit to track, analyze, and optimize your daily and weekly spending.

---

## Features

- **Add Daily Expenses:** Log your expenses in multiple categories like transport, food, data, and others.
- **Weekly Budgets:** Set weekly budgets and track your spending against it.
- **Analytics Dashboard:**
  - Key metrics: Total spend, average daily spend, weekly budget usage, and spending profile.
  - Visualizations: Daily distribution, cumulative spending, category trends, weekly patterns, and heatmaps.
  - Anomaly detection for unusual spending patterns.
- **Data Export:** Export data to CSV, Excel (multiple sheets), and PDF reports.
- **Insights:** Actionable tips based on spending patterns and trends.
- **Customizable Dashboard:** Filter by date range, theme, and analysis options.
